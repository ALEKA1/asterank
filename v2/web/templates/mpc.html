Minor Planet Center API

Overview

Asterank is happy to offer a basic queryable database for the Minor Planet Center's MPCORB.DAT data files. It is a simple way to quickly apply constraints to a set of over 600,000 asteroids. It is possible to use comparators and set specific criteria for all attributes provided by the MPC.

The database currently runs on mongodb and queries must adhere to mongo's json-influenced format for a 'find' operation. The database is updated nightly from the MPC's MPCORB.dat dataset.

Format

Requests are of the form:

http://asterank.com/api/mpc?query={query}&limit={limit}

MPC results are parsed as per the orbit format specified here. In some cases, a field may not be present in the results if it was not present in MPCORB.dat. This can be treated as an empty or unspecified attribute.

Sample Request

{% raw %} This request returns 10 asteroids with roughly circular orbits, low inclination, and semi-major axis less than 1.5 AU:

/api/mpc?query={"e":{"$lt":0.1},"i":{"$lt":4},"a":{"$lt":1.5}}&limit=10
