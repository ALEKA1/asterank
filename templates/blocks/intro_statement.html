Astrokit is an open-source project providing web-based tools for astronomers. Its primary focus is to create asteroid data analysis tools that are easy to use and widely accessible. Its goal is to enable non-professionals (students, amateur astronomers, citizen scientists) to participate in the scientific process of asteroid discovery and characterization.

The project is split into 4 main components:

  1. An easy way to browse the set of asteroids known to scientists;
  2. A simple interface to solve astrometry for any image of the night sky;
  3. A service that computes relative photometry for moving objects in a series of these images;
  4. An interface that makes it easy to view derived light curves.
